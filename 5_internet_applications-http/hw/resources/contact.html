This simple HTTP server is a demonstration project for learning network programming and HTTP protocol fundamentals.

Project Information

Course: Computer Networks

Assignment: Build a Simple HTTP Server Using Socket Programming

Technology: Python Socket Programming

Protocol: HTTP/1.1

Server Details

Default Host: 127.0.0.1 (localhost)

Default Port: 8000

Resources Directory: ./resources/

Supported Methods: GET only

How to Use

  1. Start the server: python server.py
  2. Or with custom port: python server.py 8080
  3. Or with custom host and port: python server.py 8080 0.0.0.0
  4. Open your browser and navigate to http://localhost:8000
Note: This is a simple educational HTTP server and should not be used in production environments. It lacks many features required for a production web server such as SSL/TLS support, advanced caching, compression, and robust security measures.
