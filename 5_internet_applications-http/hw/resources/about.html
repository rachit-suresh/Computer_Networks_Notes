About Simple HTTP Server

Project Overview

This is a simple HTTP server implementation built from scratch using Python socket programming. It demonstrates the fundamentals of the HTTP protocol and network communication.

Technical Details

  • Language: Python 3
  • Protocol: HTTP/1.1 over TCP
  • Architecture: Single-threaded server
  • Port: 8000 (configurable)
  • File Serving: Static HTML files from resources directory

Security Features

  • Path traversal protection using absolute path validation
  • Method restriction (only GET requests allowed)
  • Proper error handling and status codes
  • Input validation and sanitization

HTTP Status Codes Supported

200 OK - Successful file serving
400 Bad Request - Malformed requests
403 Forbidden - Unauthorized path access
404 Not Found - File doesn't exist
405 Method Not Allowed - Non-GET requests
500 Internal Server Error - Server-side errors

Learning Objectives

This project helps understand:

  • Low-level socket programming
  • HTTP protocol structure and headers
  • Web server architecture
  • Security considerations in web development
  • Error handling and logging
